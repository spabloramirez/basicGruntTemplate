# basicGruntTemplate

## Instalacion
_1. Descargar repositorio_

```
git clone https://github.com/spabloramirez/basicGruntTemplate.git 
```

_2. Instalar dependencias_

```
cd grunt
npm install 
```

## Tareas
- _Compilar sass_

```
grunt sass 
```
- _Minificar JavaScript_

```
grunt uglify 
```
- _Monitorizar cambios_

```
grunt watch
```
